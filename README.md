본 프로젝트는 지역 내 폐의약품 수거함의 위치를 쉽게 검색하고 관리할 수 있는 웹 응용 프로그램을 구현하는 것을 목표로 한다. 
사용자는 관할과 소재지를 기준으로 폐의약품 수거함을 검색하여 위치 정보를 확인할 수 있으며, 관리자는 수거함 정보를 추가, 수정, 삭제할 수 있다. 
또한 사용자는 특정 수거함의 상태에 대해 제보를 남길 수 있도록 하여, 실제 환경 변화에 따라 수거함 정보를 관리할 수 있는 구조로 설계하였다. 
본 웹 응용은 Flask 프레임워크와 SQLite 데이터베이스를 기반으로 구현되었으며, 공공데이터를 활용하여 실용성을 높였다.
